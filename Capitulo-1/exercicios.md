## Exercicios 

1.1 Suponha que você tenha uma lista com 128 nomes e esteja fazendo uma pesquisa binária. Qual seria o número máximo de etapas que você levaria para encontrar o nome desejado?

R: Log de 128 na base 2 (Quantos 2s conseguimos multiplicar para chegar a 128?) =>  7 (Número máximo de etapas para encontrar o nome)

1.2 Suponha que você duplique o tamanho da lista. Qual seria o número máximo de etapas agora?

R: 128 x 2 = 256 (Nossa lista tem o tamanho de 256); Log de 256 na base 2 (Quantos 2s conseguimos multiplicar para chegar a 256?) =>  8 (Número máximo de etapas para encontrar o nome)
